Listor

Listor är det minst vanligaste av de två vanligaste sätten att strukturera information på. Rätt Använda är de till stor hjälp för att göra sidor användbara och tillgängliga. En punkt i en lista skall vara kort annars blir det ingen lista.

<ul> - Unordered List

Elementet <ul> anger en un-ordered list, oordnad lista, och kan se ut som nedan.

  • Listan har ett indrag som standard.
  • Varje sak i listan föregås av ett punkt-tecken.
  • Användare av talsyntes hör orden list punkt eller liknande läsas upp för varje punkt.

<ol> - Ordered List

Numrerade listor eller Ordered lists, <ol>, fungerar som vanliga punktlistor men då ordningen är viktig så tilldelas varje punkt ett nummer.

  1. Det måste inte vara en decimalsk siffra.
  2. Det kan även vara bokstäver.
  3. Eller Romerska siffror.

Man kan även välja att starta listan på ett annat nummer än det första. Det gör ni med attributet <ol start="2">.

<dl> - Description List

Dessa listor används när man har namn på något och en kort beskrivning. Listtypen får användas för alla nyckelvärdesrelationer (key - value). Man måste däremot inte ha exakt jämna par mellan ordet som skall förklars, <dt>, och själva förklaringen, <dd>.

Listtypen kan användas till ordlistor, FAQ-sektioner, metadata och i princip allt som bygger på principen namn - värde.

I HTML5 kallas listtypen för description list. I HTML4 kallades den för definition list, och det namnet är fortfarande vanligt i dagligt tal. För att fungera som just en definitionslista, så behöver man dock komplettera <dt> med elementet <dfn>, enligt HTML5.

CSS för listor - <ul> och <ol>

Enligt CSS så är alla boxar som har display: list-item en lista. Det inkluderar som standard <li> och <ol> men inte definitionslistor. Listor har normalt ett punktuppställningstecken (list marker) och det har sin egen box. I allt övrigt fungerar värdet list-item som display: block.

Det finns några CSS-egenskaper just för listor:

  • list-style-position låter dig välja hur marker box förhåller sig till punktens box. Kan vara outside, inside< eller (nytt i CSS3) hanging. Standard är outside.
  • list-style-image låter dig välja vilken bild som helst som punktuppställningstecken. Standard är att inte ha någon bild alls (värdet none).
  • list-style-type låter dig välja bland ett antal färdiga tecken, exempelvis disc, circle, square - eller för numrerade listor - decimal, decimal-leading-zero, lower-roman (romerska siffror), lower-alpha, upper-alpha.
  • list-style är kortform (shorthand) för dessa tre. Kom ihåg att utelämnade egenskaper sätts till sina standardvärden

CSS3 introducerar pseudo-elementet ::marker som gör att man kan detaljstyra utseendet på marker box. Dessutom kan man göra egna list-style-type med @counter-style.

Menyer

Ett vanligt sätt att använda listor är för menyerna på webbplatser. En meny är ju en lista med länkar.
Så här kan koden se ut.

<nav>
  <ul>
    <li><a href="om-oss.html">Vilka vi är &#x3020;</a></li>
    <li><a href="arbeten.html">Vad vi gör &#x2318;</a></li>
    <li><a href="blog.html">Blogg &#x272A;</a></li>
    <li><a href="kontakt.html">Kontakt &#x260E;</a></li>
  </ul>
</nav>
